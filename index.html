Lapify.com


Developed a full-stack web application for a second-hand laptop selling platform using React for the frontend and Express.js for the backend. The frontend is built with modern tools like React Router for routing, Axios for API integration, Framer Motion for animations, and Google OAuth for secure user authentication. The backend uses Node.js with Express, connected to a MongoDB database for managing product listings and user data. The application is structured with modular code and uses environment variables to ensure scalability and maintainability. This project demonstrates proficiency in building and deploying a complete MERN stack application with real-world features like e-commerce, authentication, payment gateways, and communication APIs.

SiloDispatch


SiloDispatch is a comprehensive logistics platform we developed to manage last-mile delivery operations more efficiently. The system features a React-based admin dashboard and a React Native app for drivers. Admins can upload CSV files containing order data, which are then processed using FastAPI to perform K-Means clustering for location-based batch creation. Drivers can mark their availability, and admins can assign batches to them in real time. We used Node.js for backend API handling, and all data is stored and synced using Supabase with PostgreSQL. Additionally, the system integrates Twilio for OTP delivery verification and Razorpay for smooth UPI/COD payment reconciliation, making the entire delivery process reliable and seamless.
